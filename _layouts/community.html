The W3C Web Machine Learning Working Group collaborates with its sister Community Group that incubates related specifications to be adopted by the Working Group for standardization.

Working Group

Web Neural Network API

A dedicated low-level API for neural network inference hardware acceleration.

Community Group

WebMCP

Enabling web apps to provide JavaScript-based tools to AI agents and assistive technologies.

Community Group

Prompt API

An API for prompting browser-provided language models.

Community Group

Translator and Language Detector APIs

A set of APIs for translating text and detecting its language.

Community Group

Writing Assistance APIs

A set of APIs to help web users with writing text.

Community Group

Proofreader API

An API to help web users perform real-time proofreading of freeform text.
